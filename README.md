# Budget App

You can checkout working demo [here]("https://artemlav.github.io/budget-app/")

## Technologies used

- React
- Redux, React-Redux
- HTML5
- CSS
- Prop-types

### Dependencies
* Node v12.16.3 and higher
* NPM v6.14.4 and higher

### Installing
* Fork and clone this repository
* Run `npm install` in your terminal
* Run `npm start`
